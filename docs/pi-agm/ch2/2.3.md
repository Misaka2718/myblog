在 (2.2.5) 中，令 $x=0$ 得

$$\sqrt{s} \theta_3(e^{-s\pi}) = \theta_3(e^{-\pi/s}) \tag{2.3.1}$$

令 $x=\frac{1}{2}$ 得

$$\sqrt{s} \theta_2(e^{-s\pi}) = \theta_4(e^{-\pi/s}) \tag{2.3.2}$$

($e^{2\pi i k \frac{1}{2}} = e^{i\pi k} = (-1)^k$)

用 (2.3.2) 中替代 $s \to \frac{1}{s}$ 有

$$\sqrt{s} \theta_4(e^{-s\pi}) = \theta_2(e^{-\pi/s}) \quad \text{Re}(s) > 0 \tag{2.3.3}$$

把 (2.3.3) 和 (2.3.1) 作商

$$\frac{\theta_4(e^{-s\pi})}{\theta_3(e^{-s\pi})} = \frac{\theta_2(e^{-\pi/s})}{\theta_3(e^{-\pi/s})}$$

回忆(2.2.12)$k'(q)=\dfrac{\theta_4^2(q)}{\theta_3^2(q)}$

平方之，得 $k'(e^{-s\pi}) = k(e^{-\pi/s})$。做替换 $s \to \frac{1}{s}$有

$$k(e^{-\pi s}) = k'(e^{-\pi/s}) \tag{2.3.4}$$

由定理 2.1 $M(1, k') = \theta_3^{-2}(q), \quad k' = \theta_4^2(q)/\theta_3^2(q)$

(此处略去关于 $k$ 的说明)

此处，令 $q = e^{-\pi s}$，则

$$M(1, k'(e^{-\pi s})) = \theta_3^{-2}(e^{-\pi s})$$

代入 (2.3.4) 处与 $k$ 的联系，有

$$M(1, k(e^{-\pi s})) = \theta_3^{-2}(e^{-\pi/s})$$

将此两式作商 有

$$\pi \frac{M(1, k')}{M(1, k)} = \pi \frac{\theta_3^{-2}(e^{-\pi s})}{\theta_3^{-2}(e^{-\pi/s})} = \pi \frac{\theta_3^2(e^{-\pi/s})}{\theta_3^2(e^{-\pi s})} \overset{(2.3.1)}{=} \pi s \tag{2.3.5}$$

$$= -\ln q \quad (q = e^{-\pi s})$$

## 定理 2.3

对 $k \in (0,1)$

$$ \quad \pi \frac{M(1, k')}{M(1, k)} = -\log q, \quad k = \frac{\theta_2^2(q)}{\theta_3^2(q)}, \quad k' = \frac{\theta_4^2(q)}{\theta_3^2(q)} \tag{a}$$

且

$$ \quad \pi \frac{K(k)}{K'(k)} = -\log q \tag{b}$$

Proof. (a) 是 (2.3.5)。(b) 由 (a) 和 $K(k) = \frac{\pi}{2} \frac{1}{M(1, k')}$ 得。

(b) 常被写作 $q = e^{-\pi K'/K}$ 且 $q$ 称作 the nome associated with k。原因上 (b) 的 $q = e^{-\pi K'/K}$ 解决了取定 $k$ 所对应的 $q$ 的值。

由于 $k = \frac{\theta_2^2(q)}{\theta_3^2(q)} = \left[ \frac{\theta_2(q)}{\theta_3(q)} \right]^2$。当 $q \to 0$ 时

$$\theta_2(q) = \sum_{n=-\infty}^{\infty} q^{(n+1/2)^2} = 2q^{\frac{1}{4}} + 2q^{\frac{9}{4}} + \dots = 2q^{\frac{1}{4}} + O(q^{\frac{9}{4}})$$

$$\theta_3(q) = \sum_{n=-\infty}^{\infty} q^{n^2} = 1 + 2q + 2q^4 + \dots = 1 + O(q)$$

$$\begin{aligned} \therefore \quad \left[ \frac{\theta_2(q)}{\theta_3(q)} \right]^2 &= \left( \frac{2q^{\frac{1}{4}} + O(q^{\frac{9}{4}})}{1 + O(q)} \right)^2 = \frac{4q^{\frac{1}{2}} + 4q^{\frac{1}{4}}O(q^{\frac{9}{4}}) + O(q^{\frac{9}{2}})}{1 + 2O(q) + O(q^2)} \\ &= \frac{4\sqrt{q} + O(q^{\frac{5}{2}})}{1 + O(q)} \\ &\xlongequal{\frac{1}{1+O(q)} = 1+O(q)} (4\sqrt{q} + O(q^{\frac{5}{2}}))(1 + O(q)) \\ &= 4\sqrt{q} + O(q) \quad \text{当 } q \to 0 \end{aligned}$$

$$\text{则} \quad k = 4\sqrt{q} + O(q)$$

由于 $\frac{\pi}{M(1, k)} = \frac{-\log q}{M(1, k')}$

而

$$\log \frac{k}{4} = \log \frac{4\sqrt{q} + O(q)}{4} = \log (\sqrt{q} (1 + O(\sqrt{q})))$$


$$\begin{aligned} &= \frac{1}{2} \log q + \log(1+O(\sqrt{q})) \\ &= \frac{1}{2} \log q + O(\sqrt{q}) \end{aligned}$$

$$\text{则} \quad -\log q = 2\log \frac{4}{k} + O(\sqrt{q})$$

$$\text{从而} \quad \frac{\pi}{M(1,k)} = \frac{2\log \frac{4}{k} + O(\sqrt{q})}{M(1,k')} = [2\log \frac{4}{k} + O(\sqrt{q})] (1+\alpha)$$

$\alpha \to 0$ 当 $q \to 0$。因为 当 $q \to 0$ 时， $k \to 0^+, \quad k' \to 1$。

$$\text{而} \lim_{k' \to 1} M(1, k') = M(1, 1) = 1, \quad \text{则} \quad \frac{1}{M(1,k')} = (1+\alpha)$$

$$\frac{\pi}{2M(1,k)} = \log \frac{4}{k} + O(\sqrt{q}) + \alpha \left( \log \frac{4}{k} + O(\sqrt{q}) \right) \quad (*)$$

_(注：笔记中此处系数2已除到左边，右边括号内的2应被约去或归入 $O$)_

现在只剩 $\alpha \log \frac{4}{k}$ 的问题。由于

$$\frac{1}{M(1,k')} = \frac{2}{\pi} K(k) = \frac{2}{\pi} \int_0^{\frac{\pi}{2}} \frac{d\theta}{\sqrt{1-k^2\sin^2\theta}} < \frac{1}{\sqrt{1-k^2}}$$

_(注：这里是利用 $\sin^2\theta \le 1$ 放缩积分)_

$$\alpha = \frac{1}{M(1,k')} - 1 \le \frac{1}{\sqrt{1-k^2}} - 1$$

$$\begin{aligned} \log \frac{4}{k} \cdot \alpha &< \log \frac{4}{k} \left( \frac{1}{\sqrt{1-k^2}} - 1 \right) \\ \text{又} \quad \lim_{k \to 0^+} \log \frac{4}{k} \left( \frac{1}{\sqrt{1-k^2}} - 1 \right) &= - \lim_{k \to 0^+} \ln k \left( \frac{1}{\sqrt{1-k^2}} - 1 \right) \\ &= - \lim_{k \to 0^+} \ln k \left( \frac{k^2}{2} + O(k^4) \right) \\ &= 0 \end{aligned}$$

$$\text{则} \quad \alpha \log \frac{4}{k} \to 0 \quad \text{当} k \to 0^+ \text{了}。 \quad \text{由} (*) \text{有}$$

$$\tag{2.3.6} \quad \lim_{k \to 0^+} \left[ \frac{\pi}{2M(1,k)} - \log\left(\frac{4}{k}\right) \right] = 0$$

现在 从 $a_0=1, \ b_0=k'$ 着手 AGM 迭代。如同 2.1 所说

$$b_n/a_n = k'(q^{2^n}), \quad c_n/a_n = k(q^{2^n})$$


$$\frac{c_n}{a_n} = k(q^{2^n}) = 4\sqrt{q^{2^n}} + O(q^{2^n})$$

$$\frac{c_n}{4a_n} = \sqrt{q^{2^n}} + O(q^{2^n}) = \sqrt{q^{2^n}}(1 + O(q^{2^{n-1}}))$$

$$\therefore \quad \log \frac{c_n}{4a_n} = 2^{n-1} \log q + \log(1 + O(q^{2^{n-1}}))$$

$$= 2^{n-1} \log q + O(q^{2^{n-1}})$$

$$\therefore \quad 2^{-n} \log \frac{4a_n}{c_n} = - \frac{1}{2} \log q + O(q^{2^{n-1}})$$

$$= \frac{\pi}{2} \frac{M(1, k')}{M(1, k)} + O(q^{2^{n-1}})$$

令 $n \to \infty$ 得

$$ \quad \lim_{n \to \infty} 2^{-n} \log\left(\frac{4a_n}{c_n}\right) = \frac{\pi}{2} \frac{M(1, k')}{M(1, k)} \tag{2.3.7}$$

现在 对 (2.3.7) 对 $s$ 求导，交换极限和求导顺序

$$\frac{d}{ds} \lim_{n \to \infty} 2^{-n} \left( \log 4 + \log \frac{a_n}{c_n} \right) = \frac{d}{ds} \frac{\pi}{2} \frac{M(1, k')}{M(1, k)} = \frac{d}{ds} \left( \frac{\pi}{2} s \right)$$

$$\lim_{n \to \infty} \frac{d}{ds} \left( 2^{-n} \log \frac{a_n}{c_n} \right) = \frac{\pi}{2}$$

则

$$\text{令} \quad \pi_n := 2^{1-n} \frac{d}{ds} \log\left(\frac{a_n}{c_n}\right) \to \pi, \quad n \to \infty$$

**现证明**(d表示微分)

$$ \quad 2^{-n} b_n^{-2} d \log \frac{a_n}{c_n} = b_0^{-2} d \log \frac{a_0}{c_0} \tag{2.3.8}$$

书本用来留作习题…… 你觉得有可能做得出吗？作者.要证明上式子成立，相当
证明$\quad 2^{-n} b_n^{-2} d \log \frac{a_n}{c_n}$是与$n$的无关的量：

$$2^{-(n+1)} b_{n+1}^{-2} d \log \frac{a_{n+1}}{c_{n+1}} = 2^{-n} b_n^{-2} d \log \frac{a_n}{c_n}$$

或约去 $2^{-n}$， $b_{n+1}^{-2} d \log \frac{a_{n+1}}{c_{n+1}} = 2 b_n^{-2} d \log \frac{a_n}{c_n}$

$$\frac{c_{n+1}}{a_{n+1}} = \frac{\frac{1}{2}(a_n-b_n)}{\frac{1}{2}(a_n+b_n)} = \frac{1-k_n'}{1+k_n'} \quad (\text{此处 } k_n' = \frac{b_n}{a_n})$$

$$\begin{aligned} d \log \frac{a_{n+1}}{c_{n+1}} &= d(\ln(1+k_n') - \ln(1-k_n')) \\ &= \frac{dk_n'}{1+k_n'} - \frac{-dk_n'}{1-k_n'} = \left( \frac{1}{1+k_n'} + \frac{1}{1-k_n'} \right) dk_n' \\ &= \frac{2}{1-k_n'^2} dk_n' = \frac{2}{k_n^2} dk_n' \end{aligned}$$

$$\begin{aligned} \text{则 } b_{n+1}^{-2} d \log \frac{a_{n+1}}{c_{n+1}} &= \frac{1}{a_n b_n} \frac{2}{k_n^2} dk_n' = \frac{1}{a_n(a_n k_n')} \frac{2}{k_n^2} dk_n' \\ &= \frac{2}{a_n^2 k_n' k_n^2} dk_n' \end{aligned}$$

$$\begin{aligned} \text{左端 } 2 b_n^{-2} d \log \frac{a_n}{c_n} &= \frac{2}{b_n^2} (-1) d \log k_n \\ &= \frac{2}{a_n^2 k_n'^2} (-1) \frac{dk_n}{k_n} = - \frac{2}{a_n^2 k_n'^2 k_n} dk_n \end{aligned}$$

**故要证**

$$\frac{2}{a_n^2 k_n' k_n^2} dk_n' = - \frac{2}{a_n^2 k_n'^2 k_n} dk_n \quad \text{约去 } \frac{2}{a_n^2 k_n k_n'}$$

$$\frac{dk_n'}{k_n} = - \frac{dk_n}{k_n'}$$

即 $k_n' dk_n' + k_n dk_n = 0$

即 $\frac{1}{2} d(k_n^2 + k_n'^2) = 0$。 又 $k_n^2 + k_n'^2 = 1$。这是成立的。

**则有**

$$2^{-(n+1)} b_{n+1}^{-2} d \log \frac{a_{n+1}}{c_{n+1}} = 2^{-n} b_n^{-2} d \log \frac{a_n}{c_n} = \dots = 2^0 b_0^{-2} d \log \frac{a_0}{c_0}$$

（蓝字批注：$2^{-n} b_n^{-2} d \log \frac{a_n}{c_n}$ 是不变量）

$$\tag{2.3.8} \quad 2^{-n} b_n^{-2} d \log \frac{a_n}{c_n} = b_0^{-2} d \log \frac{a_0}{c_0} \quad ( \frac{a_0}{c_0} = \frac{1}{k})$$

**则由式**

$$2^{1-n} \frac{d}{ds} \log \frac{a_n}{c_n} = 2 \frac{b_n^2}{b_0^2} \frac{d}{ds} \log \frac{a_0}{c_0} = -2 \frac{b_n^2}{b_0^2} \frac{1}{k} \frac{dk}{ds}\left( = \frac{b_n^2}{b_0^2} (-\frac{2}{k} \frac{dk}{ds}) \right)$$

_(注：这里是将 (2.3.8) 两边同乘 $2b_n^2$ 并对 $s$ 求导，左边凑出 $\pi_n$ 的定义)_

$$\therefore \quad \pi_n = \frac{b_n^2 \pi_0}{b_0^2} \quad \text{其中 } \pi_0 = - \frac{2}{k} \frac{dk}{ds} \quad $$

在 $\pi_n = \frac{b_n^2}{b_0^2} \pi_0$ 两侧取极限， $\pi_n \to \pi, \quad b_n \to M(1, k')$

$$\text{有 } \quad \pi = \frac{M^2(1, k')}{k'^2} \left(-\frac{2}{k} \frac{dk}{ds}\right)， \text{整理得}$$

_(注：此处 $b_0 = k'$)_

$$(2.3.9) \quad \frac{dk}{ds} = - \frac{\pi}{2} \frac{k k'^2}{M(1, k')^2} = - \frac{2}{\pi} k k'^2 K^2 \tag{2.3.9}$$

_(注：这里用了 $K = \frac{\pi}{2} \frac{1}{M(1,k')}$)_

又 $s = -\pi^{-1} \log q, \quad \frac{dk}{dq} = \frac{dk}{ds} \frac{ds}{dq} = \frac{dk}{ds} \frac{(-\pi)^{-1}}{q}$，有

$$(2.3.10) \quad \frac{dk}{dq} = \frac{1}{2q} \frac{k k'^2}{M(1, k')^2} = \frac{2k k'^2 K^2}{q \pi^2} \tag{2.3.10}$$

回忆起 $k = \frac{\theta_2^2}{\theta_3^2}, \quad k' = \frac{\theta_4^2}{\theta_3^2}, \quad K = \frac{\pi}{2} \theta_3^2$，代入 (2.3.9)

$$\frac{dk}{ds} = \left( \frac{\theta_2^2}{\theta_3^2} \right)' = 2 \frac{\theta_2}{\theta_3} \frac{\dot{\theta}_2 \theta_3 - \theta_2 \dot{\theta}_3}{\theta_3^2}$$

_(注：此处求导是对 $s$ 求导，$\dot{\theta}$ 表示 $d\theta/ds$)_

$$- \frac{2}{\pi} k k'^2 K^2 = - \frac{2}{\pi} \frac{\theta_2^2}{\theta_3^2} \frac{\theta_4^4}{\theta_3^4} \frac{\pi^2}{4} \theta_3^4 = - \frac{\pi}{2} \frac{\theta_2^2}{\theta_3^2} \theta_4^4$$

$$2 \frac{\theta_2}{\theta_3} \frac{\dot{\theta}_2 \theta_3 - \theta_2 \dot{\theta}_3}{\theta_3^2} = - \frac{\pi}{2} \frac{\theta_2^2}{\theta_3^2} \theta_4^4 \quad \text{约去 } (-1) \text{ 和 } \frac{\theta_2}{\theta_3}$$

$$2 \frac{\theta_2 \dot{\theta}_3 - \dot{\theta}_2 \theta_3}{\theta_3^2} = + \frac{\pi}{2} \frac{\theta_2}{\theta_3} \theta_4^4$$

整理得 (2.3.11) 有

$$\frac{\dot{\theta}_3}{\theta_3} - \frac{\dot{\theta}_2}{\theta_2} = \frac{\pi}{4} \theta_4^4 \quad \text{(w.r.t. s) *} \tag{2.3.11}$$

_(蓝字注：_ 指对 $s$ 求导)*

对 (2.3.1) $\sqrt{s} \theta_3(s) = \theta_3(s^{-1})$ 两边求导：

$$\frac{1}{2\sqrt{s}} \theta_3(s) + \sqrt{s} \dot{\theta}_3(s) = - \frac{1}{s^2} \dot{\theta}_3(s^{-1}) \quad \text{两边乘 } s\sqrt{s} \text{ 有}$$

$$ \quad s^2 \dot{\theta}_3(s) + \frac{s}{2} \theta_3(s) = - s^{-1/2} \dot{\theta}_3(s^{-1}) \quad (w.r.t. s) \tag{2.3.12}$$

对 (2.3.2) 求导 $\sqrt{s} \theta_4(s) = \theta_2(s^{-1})$ 同理：

$$\frac{1}{2\sqrt{s}} \theta_4(s) + \sqrt{s} \dot{\theta}_4(s) = - \frac{1}{s^2} \dot{\theta}_2(s^{-1})$$

$$\quad s^2 \dot{\theta}_4(s) + \frac{s}{2} \theta_4(s) = - s^{-1/2} \dot{\theta}_2(s^{-1}) \quad (w.r.t. s) \tag{2.3.13}$$


(2.3.12) 与 (2.3.1) 作商有

$$\frac{1}{2s} + \frac{\dot{\theta}_3}{\theta_3}(s) = - \frac{1}{s^2} \frac{\dot{\theta}_3}{\theta_3}(s^{-1}) \quad \text{乘上 } s, \text{移项}$$

$$s \frac{\dot{\theta}_3}{\theta_3}(s) + \frac{1}{s} \frac{\dot{\theta}_3}{\theta_3}(s^{-1}) = - \frac{1}{2}$$

对 (2.3.13) 与 (2.3.2) 作商亦有

$$\frac{\dot{\theta}_4}{\theta_4}(s) + s^{-1} \frac{\dot{\theta}_2}{\theta_2}(s^{-1}) = - \frac{1}{2s}$$

_(注：此处推导过程稍微跳跃，实际上是利用了 $(s^2\dot{\theta}_4 + s/2 \theta_4) / (\sqrt{s}\theta_4) = (-s^{-1/2}\dot{\theta}_2)/\theta_2$，化简后两边乘 $s^{-1}$ 得到上式)_

则得

$$ \quad s \frac{\dot{\theta}_3}{\theta_3}(s) + s^{-1} \frac{\dot{\theta}_3}{\theta_3}(s^{-1}) = - \frac{1}{2} = s \frac{\dot{\theta}_4}{\theta_4}(s) + s^{-1} \frac{\dot{\theta}_2}{\theta_2}(s^{-1}) \tag{2.3.14}$$

又 (2.3.2) 有 $\sqrt{s} \theta_4(s) = \theta_2(s^{-1})$。 平方之 又 平方之 $s^2 \theta_4^4(s) = \theta_2^4(s^{-1})$。

(注：笔记右侧批注指出逆变换关系) $\theta_4^4(s^{-1}) = s^2 \theta_2^4(s)$

在 (2.3.14) 移项有

$$s \frac{\dot{\theta}_4}{\theta_4}(s) - s \frac{\dot{\theta}_3}{\theta_3}(s) = s^{-1} \frac{\dot{\theta}_3}{\theta_3}(s^{-1}) - s^{-1} \frac{\dot{\theta}_2}{\theta_2}(s^{-1})$$

利用 2.3.11 ($\frac{\dot{\theta}_3}{\theta_3} - \frac{\dot{\theta}_2}{\theta_2} = \frac{\pi}{4}\theta_4^4$，变量为 $s^{-1}$)

$$= s^{-1} \left[ \frac{\pi}{4} \theta_4^4(s^{-1}) \right] = \frac{\pi}{4} \frac{1}{s} s^2 \theta_2^4(s) = \frac{\pi}{4} s \theta_2^4(s)$$

两边约去 $s$，有

$$ \quad \frac{\dot{\theta}_4}{\theta_4} - \frac{\dot{\theta}_3}{\theta_3} = \frac{\pi}{4} \theta_2^4 \quad (w.r.t. s) \tag{2.3.15}$$

将 (2.3.11) 与 (2.3.15) 相加得

$$ \quad \frac{\dot{\theta}_4}{\theta_4} - \frac{\dot{\theta}_2}{\theta_2} = \frac{\pi}{4} (\theta_2^4 + \theta_4^4) = \frac{\pi}{4} \theta_3^4 \quad (w.r.t. s) \tag{2.3.16}$$

这里用上 (2.3.10) $\theta_2^4 + \theta_4^4 = \theta_3^4$

现在用 theta 函数表示 $E$。($K$ 已知为 $\frac{\pi}{2} \theta_3^2$)


由 $kk'^2 \frac{dK}{dk} = E - k'^2 K$，有 $E = kk'^2 \frac{dK}{dk} + k'^2 K$

$$\therefore \quad E - K = kk'^2 \frac{dK}{dk} - k^2 K $$

在 (2.3.9) $\frac{dk}{ds} = - \frac{2}{\pi} kk'^2 K^2$ 中，~~两边同乘~~ $\frac{dK}{dk}$

(注：此处笔记有涂改，意在利用链式法则)

$$\frac{dk}{ds} \frac{dK}{dk} = \frac{dK}{ds}$$

$$\text{因此} \quad kk'^2 \frac{dK}{dk} = - \frac{\pi}{2} \frac{1}{K^2} \frac{dK}{ds}$$

_(注：由 (2.3.9) 移项得 $kk'^2 = -\frac{\pi}{2K^2}\frac{dk}{ds}$，代入上式)_

$$E - K = - \frac{\pi}{2} \frac{1}{K^2} \frac{dK}{ds} - k^2 K$$

$$= - \frac{\pi}{K} \left( \frac{1}{2K} \frac{dK}{ds} + \frac{k^2 K^2}{\pi} \right)$$

又 $K = \frac{\pi}{2} \theta_3^2(q) = \frac{\pi}{2} \theta_3^2(s)$，则

$$\frac{dK}{ds} = \pi \theta_3(s) \dot{\theta}_3(s) \quad \text{，两边同除 } 2K = \pi \theta_3^2(s)$$

$$\frac{1}{2K} \frac{dK}{ds} = \frac{\dot{\theta}_3}{\theta_3}$$

又

$$\frac{k^2 K^2}{\pi} = \frac{1}{\pi} \frac{\theta_2^4}{\theta_3^4} \frac{\pi^2}{4} \theta_3^4 = \frac{\pi}{4} \theta_2^4$$

$$\therefore \quad E - K = - \frac{\pi}{K} \left( \frac{\dot{\theta}_3}{\theta_3} + \frac{\pi}{4} \theta_2^4 \right)$$

分步代入 $\frac{\dot{\theta}_4}{\theta_4} = \frac{\dot{\theta}_3}{\theta_3} + \frac{\pi}{4} \theta_2^4$ (来自 2.3.15)，与 $K = \frac{\pi}{2} \theta_3^2$

$$  E = K - \frac{\pi}{K} \frac{\dot{\theta}_4}{\theta_4} \tag{2.3.17}$$

$$\text{或者} \quad E = \frac{\pi}{2} \theta_3^2 - 2 \frac{\dot{\theta}_4}{\theta_3^2 \theta_4}$$

关于 $K'$ 呢 
$$K'(k) = \frac{\pi}{2} \frac{1}{M(1,k)} = \frac{\pi}{2} \theta_3^2(e^{-\pi/s}) = \frac{\pi}{2} \theta_3^2(s^{-1})$$

见 (2.3.4) 与 (2.3.5) 之间那式

$$= \frac{\pi}{2} s \theta_3^2(s) = sK$$

现在求 $E'$，由之前上章 $\frac{dK'}{dk} = \frac{k^2 K' - E'}{kk'^2}$，如法炮制

$$E' - K' = k^2 K' - kk'^2 \frac{dK'}{dk} - K'$$

$$= -kk'^2 \frac{dK'}{dk} - k'^2 K'$$

又以 $\frac{dK'}{dk}$ 乘 $\frac{dk}{ds} = -\frac{2}{\pi} kk'^2 K^2$ 有

$$\frac{dK'}{ds} = - \frac{2}{\pi} kk'^2 K^2 \frac{dK'}{dk}$$

$$\therefore kk'^2 \frac{dK'}{dk} = - \frac{\pi}{2} \frac{1}{K^2} \frac{dK'}{ds} \quad \text{代入 } E' - K' \text{ 有}$$

$$E' - K' = + \frac{\pi}{2} \frac{1}{K^2} \frac{dK'}{ds} - k'^2 K'$$

$$= \frac{\pi}{K} \left( \frac{1}{2K} \frac{dK'}{ds} - \frac{k'^2 K' K}{\pi} \right)$$

又

$$\frac{dK'}{ds} = \pi \theta_3(s^{-1}) \dot{\theta}_3(s^{-1}) (-s^{-2})$$

$$2K = \pi \theta_3^2(s) = \pi \theta_3^2(s^{-1}) s^{-1}$$

$$\therefore \frac{1}{2K} \frac{dK'}{ds} = - \frac{1}{s} \frac{\dot{\theta}_3}{\theta_3}(s^{-1}) \quad \text{，由 (2.3.14) 有}$$


$$= \frac{1}{2} + s \frac{\dot{\theta}_3}{\theta_3}$$

又

$$\frac{k'^2 K K'}{\pi} = \frac{1}{\pi} \frac{\theta_4^4}{\theta_3^4} \cdot \frac{\pi}{2} \theta_3^2 \cdot \frac{\pi}{2} s \theta_3^2 = \frac{\pi}{4} s \theta_4^4$$

$$\therefore \frac{1}{2K} \frac{dK'}{ds} - \frac{k'^2 K' K}{\pi} = \frac{1}{2} + s \frac{\dot{\theta}_3}{\theta_3} - \frac{\pi}{4} s \theta_4^4 \xlongequal{2.3.11} \frac{1}{2} + s \frac{\dot{\theta}_2}{\theta_2}$$

$$\begin{aligned}
\therefore E' - K'& = \frac{\pi}{K} \left( \frac{1}{2} + s \frac{\dot{\theta}_2}{\theta_2} \right)\\
&= \frac{\pi}{2K} + \frac{\pi s}{K} \frac{\dot{\theta}_2}{\theta_2}, \quad \text{又 } \pi s = -\log q = \pi \frac{K'}{K}\\
&= \frac{\pi}{2K} + \frac{1}{K} \pi \frac{K'}{K} \frac{\dot{\theta}_2}{\theta_2}\\
&= \frac{\pi}{2K} + \frac{\pi K'}{K^2} \frac{\dot{\theta}_2}{\theta_2} \quad (w.r.t. s) \quad\quad(2.3.18a)
\end{aligned}
$$

$$
\begin{aligned}
\text{又 } \frac{\dot{\theta}_2}{\theta_2} &= \frac{\dot{\theta}_4}{\theta_4} - \frac{\pi}{4} \theta_3^4\\
&= \frac{\dot{\theta}_4}{\theta_4} - \frac{\pi}{4} \frac{4K^2}{\pi^2}\\
&= \frac{\dot{\theta}_4}{\theta_4} - \frac{1}{\pi} K^2
\end{aligned}
$$

$$\therefore \frac{\pi K'}{K^2} \frac{\dot{\theta}_2}{\theta_2} = \frac{\pi K'}{K^2} \frac{\dot{\theta}_4}{\theta_4} - K'$$

(2.3.18a) 可变为 $E' - K' = \frac{\pi K'}{K^2} \frac{\dot{\theta}_4}{\theta_4} - K' + \frac{\pi}{2K}$

$$E' = \frac{\pi}{2K} + \frac{\pi K'}{K^2} \frac{\dot{\theta}_4}{\theta_4} \quad (w.r.t. s) \tag{2.3.18b}$$
